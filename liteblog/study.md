```
## Go 升级,通过该方式用户可以升级 beego 框架，强烈推荐该方式：
go get -u github.com/astaxie/beego

## 本地执行安装就可以升级了
go install  github.com/astaxie/beego

export GO111MODULE=on
export GOPROXY=https://goproxy.io // 设置代理

```

- beego运行命令: bee run
- 查看已经安装的依赖包：go list -m al
